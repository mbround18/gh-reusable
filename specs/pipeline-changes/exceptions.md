# Pipeline Compliance Exceptions

Use this registry for approved, time-bound compliance waivers.

## Active exceptions

| ID     | Spec | Pipeline Ref | Rule | Owner | Approved By | Expires At | Remediation Plan | Status |
| ------ | ---- | ------------ | ---- | ----- | ----------- | ---------- | ---------------- | ------ |
| _none_ | -    | -            | -    | -     | -           | -          | -                | -      |

## Rules

1. Exceptions must be temporary and linked to a specific pipeline-change spec.
2. Owner, expiration, and remediation plan are mandatory.
3. Expired exceptions are invalid and must not be used to waive checks.
